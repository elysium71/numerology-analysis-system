## Description
This project calculates a person's lucky number based on their birth date, determines their lucky animal, and checks if their number is a master number. It also categorizes the person into a generation based on their birth year.

## Features
- Calculate lucky number
- Determine lucky animal
- Check for master numbers
- Determine generation based on birth year
- Compare lucky numbers and animals between two birthdates

## Running the Program
To run the program, use the following command:
   python3 main.py

## Requirements
Python 3.x \
Linux Environment
